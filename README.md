# Doogee-T30-Pro-Repair
Battery is too low - не заряджається планшет
перевірка обвязки на пробій
* **SW (Switching Node): Це вихід на котушку. Якщо там після пайки буде КЗ на землю — значить, сів криво або злипнулися кулі під процесором.**
* **VBUS: Вхід від зарядки. Має дзвонитися так само, як ти міряв раніше (571 у діодній прозвонці).**
* **VBAT: Вихід на акум. Твої магічні "396" мають повернутися.**
* **SCL/SDA: Шина даних. Якщо чіп не сяде на ці п'ятаки, планшет його просто "не побачить", і зарядки не буде, хоча старт може бути.**
* Для ліній SDA / SCL (піни 12 і 13) порядок такий:Червоний щуп — на корпус (GND) або будь-який металевий екран плати.
Чорний щуп — на сам пін 12-13 чіпу sc89890h або на контрольні точки (резистори підтяжки) поруч із чіпом.
Що ти маєш побачити:
На обох лініях (SDA і SCL) значення мають бути майже однаковими (зазвичай у межах 400-700мВ).
Якщо на одному піні 500, а на іншому 020 або O (нескінченність) — значить, або чіп не сів на цей п'ятак, або під ним "сопля", або лінія обірвана.
Важливо: Оскільки ці лінії шини зазвичай підтягнуті через резистори до напруги 1.8V або 3.3V, дзвонити їх треба на вимкненій платі без акумулятора.
